# Golden Examples

These examples are the polished first-run paths for the three common
LogicPearl artifact shapes:

- [Gate](./gate-approve-deny): approve or deny a request
- [Action](./action-next-step): choose one next step
- [Fan-out](./fanout-applicable-actions): list every applicable action

Each directory includes reviewed traces, a feature dictionary, a runnable input,
a changed trace set for `diff`, and a README with build, inspect, run, compile,
native, browser, and verify commands.
